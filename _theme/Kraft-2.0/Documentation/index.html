Installation

Below are the steps to get started with your Site Template.

  1. Open the Kraft Folder to find all the Templates Files.
  2. You will need to Upload all the file from HTML folder to your Web Server using FTP.
  3. Below are the required files/folders to be upload on the Web Server:
    • HTML/required HTML files
    • HTML/ajax-projects
    • HTML/assets/css
    • HTML/assets/images
    • HTML/assets/js
    • HTML/assets/lib
    • HTML/include
    • HTML/style.css
    You can add new file/folder according to your preferences.

Favicons

Set the Favicon of your Website in head section as given below:

<link rel="icon" type="image/png" href="assets/images/favicon.png">

Logo Settings

Set the logo inside id="site-branding" container as shown below :

<div id="site-branding">
<a  href="#">
        <img class="logo" src="" alt="Logo">					                            
        <img class="retina-logo" src="" alt="Retina Logo">					                            
</a>
</div>

Changing Fonts

Change the font in the head section, if you want to use a Google Font.

<link href="https://fonts.googleapis.com/css?family=Roboto:300,400,500,700" rel="stylesheet"/>

After the font is changed in link tag, change the font name in style.css

Settings

  • data-layoutmode- Set the layout for portfolio. Eg. grid/mosaic
  • data-gaphorizontal- Horizontal space between each item. Eg. 30 (this will be 30px margin)
  • data-gapvertical- Vertical space between each item. Eg. 30 (this will be 30px margin)
  • data-captionanimation- Animation occurs when the cursor is hovered on portfolio thumbnail. Eg. zoom
  • data-animationtype- Animation occurs when portfolio items are filtered Eg. quicksand

List of available hover effects for portfolio item.

Load More Items

There are 2 options to load more items in the portfolio section.

  1. On button click ,data-loadmoreaction="click"
  2. On hover ,data-loadmoreaction="auto"

In both the case, the items to load will be in the different HTML file called loadMore.html located in ajax-projects.

Note Load more feature will not work without the server. It will only run on server, be it on a local server or a web server.

Google Map Setup

To use a google map, API is required. You will need to generate one if you don't have. You will need to replace the API in scripttag where it says "yourapikey". Snippet is given below.

<script type="text/javascript" src="//maps.googleapis.com/maps/api/js?key=yourapikey"></script>

Click here for the information on generating API key.

Customisation

If you want to do any kind of customization in css or javascript, the best way would be to create new custom file. Your custom file would be the last file linked in the document. css file would go in the head section and javascript file would go at the end of the document where other scripts are linked, before the end of body tag.
